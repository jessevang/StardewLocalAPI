Stardew Local API (Developer Toolkit) is a local web UI for building and testing Stardew Valley mods.
Tools overview
Icon Tool What it does Typical use
🏠 Home Overview explains what each tool does. Overview of each tool.
🧩 Event Builder Build events, Run events, Save Event Progress, and output as content patcher mod. Other feature includes autopopulating dropdowns, map picker for move and warp commands, commands and Loading and or Saving existings presets command step. Save and Load Builds and test events
📦 Export Mod Package your project into a ready-to-use Content Patcher mod. Generates the manifest.json, and content.json files based on your current project. Generate and download mod files
▶️ Event Player Loads events and run them in-game without checking preconditions. Search to Play or Stop in game events
🗣️ Dialogue Player Search dialogue lines and play them on demand. Search to Play or Stop in game Dialogue.
🎵 Music Player Search and play music cues. Sorts In Events and played in Location. Search to Play or Stop in game Music.
🔊 Sound Player Search and play sounds. Search to Play or Stop in game sounds.
