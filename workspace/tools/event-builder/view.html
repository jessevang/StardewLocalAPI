Run Event

Runs the event script directly in-game.
Ignores event preconditions and starts immediately.

Best for rapid event testing while building.
Run via Content Patcher

Generates a Content Patcher files to replace "[CP] StardewLocalAPI Dev".

The content patcher's reload command is copied automatically.
Paste it into the SMAPI console and press enter to reload the pack.

Click Run CP again to test the event with its preconditions required.
End Current Event

Ends the event that is currently running in-game.

Use this if an event gets stuck, loops, or you want to stop testing early.

I18N Export

When enabled, dialogue text in the event is exported to an i18n/default.json file instead of being written directly into the event script.

Applies to Run CP and Build Preview.
Split CP Preview

When enabled, Build Preview shows the Content Patcher output as separate files: content.json and assets/data/events/events.json.

Applies to Build Preview, Copy, Download, and Run CP.
